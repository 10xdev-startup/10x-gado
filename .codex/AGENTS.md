## gstack (recommended)

This project uses [gstack](https://github.com/garrytan/gstack) for AI-assisted workflows.
Install it for the best experience:

```bash
git clone --depth 1 https://github.com/garrytan/gstack.git ~/.claude/skills/gstack
cd ~/.claude/skills/gstack && ./setup --host codex
```

Skills like /gstack-qa, /gstack-ship, /gstack-review, /gstack-investigate, and /gstack-browse
become available after install. Use /gstack-browse for all web browsing.
Use ~/.claude/skills/gstack/... for gstack file paths.
